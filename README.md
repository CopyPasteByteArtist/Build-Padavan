# Build Padavan

Padavan for MediaTek MT7620/MT7621-based routers <br>

能够在MediaTek MT7620/MT7621处理器的路由器上运行的Padavan <br>

遵循原汁原味的Padavan，配置文件在压缩包中，名称是`<机型名>.config` <br>

默认Web登录IP为原设备出厂IP <br>

默认Web用户名/密码：admin/admin，更多信息在`trunk/user/shared/defaults.h` <br>
